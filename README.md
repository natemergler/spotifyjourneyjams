# spotifyjourneyjams

## Installation
Download main.py, then
```
pip install spotipy
```
and you are all set. Just make a spotify developer account and set your environmental variables for your client secret, client id, and redirect uri.
On windows paste below into PowerShell
```
$env:SPOTIPY_CLIENT_ID='your-spotify-client-id'
$env:SPOTIPY_CLIENT_SECRET='your-spotify-client-secret'
$env:SPOTIPY_REDIRECT_URI='your-app-redirect-url'
```
## Use
Run the script and it will walk through everything step by step.
